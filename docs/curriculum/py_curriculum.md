- 4.11 ลูปทำงานซ้ำ while
  - 4.11.1 โครงสร้างคำสั่ง while loop
  - 4.11.2 การควบคุมลูปด้วย break และ continue
    - ตัวอย่างการใช้งาน `break` ใน `while` loop เพื่อหยุดการทำงานของลูปทันที:
      ```python
      # แสดงผลเลข 1 ถึง 5 แต่หยุดทำงานเมื่อพบเลข 3
      count = 1
      while count <= 5:
          if count == 3:
              break  # หลุดออกจากลูปทันที
          print("รอบที่:", count)
          count += 1
      # ผลลัพธ์:
      # รอบที่: 1
      # รอบที่: 2
      ```
    - ตัวอย่างการใช้งาน `continue` ใน `while` loop เพื่อข้ามการทำงานรอบปัจจุบันไปรอบถัดไป:
      ```python
      # แสดงผลเลข 1 ถึง 5 แต่ข้ามการแสดงผลเมื่อพบเลข 3
      count = 0
      while count < 5:
          count += 1
          if count == 3:
              continue  # ข้ามคำสั่ง print ด้านล่างแล้วกลับไปเช็คเงื่อนไขทันที
          print("รอบที่:", count)
      # ผลลัพธ์:
      # รอบที่: 1
      # รอบที่: 2
      # รอบที่: 4
      # รอบที่: 5
      ```
- 4.12 การวนซ้ำด้วย for และฟังก์ชัน range()
  - 4.12.1 โครงสร้างคำสั่ง for loop

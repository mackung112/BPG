## 📋 บันทึกส่งมอบ (เก็บ 3 รายการล่าสุด)

- **ปรับแก้เนื้อหา 31910-0001 เลื่อนบทที่ 3.1 เป็น 3.2 และสร้าง 3.1 ใหม่เรื่องการดาวน์โหลด Windows 11** [26/06/2026]
  - แก้ไขไฟล์ `itCourse.js` ปรับรหัส `3.1` (การติดตั้งระบบปฏิบัติการ) เป็น `3.2` และสร้าง `3.1` ใหม่ "การเตรียมไฟล์ติดตั้ง (ดาวน์โหลด Windows 11)"
  - ย้ายไฟล์ `it3_1.jsx` เดิมไปเป็น `it3_2.jsx` เพื่อเก็บเนื้อหาเดิมไว้ครบถ้วน
  - สร้างไฟล์ `it3_1.jsx` ใหม่ โดยจำลองหน้าเว็บดาวน์โหลด Windows 11 (Glassmorphism & Lucide) พร้อมคำอธิบายตัวเลือก 3 รูปแบบ (Installation Assistant, Installation Media, ISO File)

- **ลบหัวข้อที่ไม่มีเนื้อหาในวิชาการเขียนโปรแกรมเชิงวัตถุ (OOP)** [26/06/2026]
  - ลบหัวข้อบทเรียนใน `oopCourse.js` ทั้งหมดที่ไม่มีเนื้อหาและขึ้นตัวสำรอง (placeholder) อยู่ระหว่างจัดเตรียม
  - ลบหน่วยการเรียนรู้ที่ไม่มีหัวข้อเหลืออยู่เลยออกทั้งหมด (หน่วยที่ 4 และหน่วยที่ 5) เพื่อให้เมนูด้านข้างสะอาดและยอดหน่วยการเรียนรู้นับได้ถูกต้อง
  - ตรวจสอบความถูกต้องด้วยการรัน `npx impeccable detect` และรัน `npm run build` ผ่าน 100%

- **สร้างหน้าคู่มือสอน Python: OOP & Tkinter (Modern Design) ในบทเรียน 3.1** [26/06/2026]
  - นำ HTML โค้ดของคู่มือสอนไปสร้างเป็น React Component (oop3_1.jsx) พร้อมฝัง CSS สไตล์ทั้งหมด
  - เชื่อมโยงบทเรียน `3.1` ในไฟล์ `oopCourse.js` ให้แสดงผลเป็น `[oop3_1]` และตั้งค่า `hideHeader: true` เพื่อใช้ Header จากดีไซน์ต้นฉบับ
  - ตรวจสอบคุณภาพด้วย `npx impeccable detect` ผ่านสมบูรณ์

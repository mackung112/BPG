# การตรวจสอบคุณภาพ

---

## บังคับทุกครั้งก่อนจบงาน

1. รัน `npx impeccable detect --gemini <path>` ก่อนจบงานทุกครั้ง
2. รัน `npm run build` ตรวจว่า build ผ่าน 100%

## กฎ UI ที่ต้องตรวจ

- หลีกเลี่ยงคู่สี AI เช่น ม่วง/น้ำเงิน Gradient สลับเรืองแสง
- ใช้ Glassmorphism + ขอบบาง `border-l-[3px]` แทนขอบทึบ `border-l-4`
- ห้ามตัวอักษรสีเทาบนพื้นสีอิ่ม (เช่น `text-slate-500` บน `bg-emerald-50`)
- ห้ามแอนิเมชันเด้ง `animate-bounce` — ใช้ `ease-out` นุ่มนวลแทน

## ขนาดตัวอักษรที่ถูกต้อง

- เนื้อหาหลัก: 16-17px
- คำอธิบายย่อย: 13-14px
- โค้ด: 13.5px
- เส้นเชื่อม SVG: เข้า-ออกจากจุดกึ่งกลางโมดูล 100% เป็นเส้นฉาก 90°

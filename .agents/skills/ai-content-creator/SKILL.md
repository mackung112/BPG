# คู่มือสร้างเนื้อหาบทเรียน

---

## ขั้นตอนการสร้าง

1. **รับข้อมูลจากผู้ใช้**: ชื่อวิชา, รหัสวิชา, หัวข้อบท, เนื้อหาที่ต้องการ
2. **สร้าง React Component** ที่ `src/components/interactive/<วิชา>/<prefix><X>_<Y>.jsx`
3. **เพิ่มข้อมูลใน Course Data**:
   - หากเป็นวิชาใหม่ → สร้างไฟล์ `src/data/<prefix>Course.js` + เพิ่ม import ใน `data.js`
   - หากเป็นวิชาเดิม → เพิ่ม lesson ในไฟล์ course ที่มีอยู่
4. **ใส่ Marker** `[<prefix><X>_<Y>]` ในฟิลด์ `content` ของบทเรียน

---

## โครงสร้างข้อมูลวิชาใหม่

```javascript
const newCourse = {
    id: "รหัสวิชา",           // เช่น "21910-1003"
    level: "ปวช.",            // "ปวช." ถ้ารหัสขึ้นต้น 2, "ปวส." ถ้ารหัสขึ้นต้น 3
    title: "ชื่อวิชาภาษาไทย",
    description: "คำอธิบายรายวิชา",
    icon: "🐍",               // Emoji ที่เหมาะกับวิชา
    chapters: [
        {
            id: 1,
            title: "หน่วยที่ 1 ชื่อหน่วย",
            lessons: [
                {
                    id: "1.1",
                    title: "1.1 ชื่อบทเรียน",
                    mainTitle: "ชื่อบทเรียน (ไม่มีเลขนำหน้า)",
                    subTitle: "(English Title)",
                    description: "คำอธิบายบทเรียน",
                    content: "[prefix1_1]"
                }
            ]
        }
    ]
};
```

## กฎการสร้างเนื้อหา

- **ทฤษฎีภาษาไทย** — อธิบายลึกซึ้งครบทุกหัวข้อย่อย ห้ามละเว้นหรือยุบรวม
- **Simulator ไม่บังคับ** — มีได้จะดี AI ประเมินเองว่าหัวข้อไหนเหมาะจะมี Interactive
- **เรียงซ้อนแนวดิ่ง** — ทุกหัวข้อย่อยเรียงต่อกันในไฟล์เดียว ห้ามแยกหน้า/แท็บ
- **Immersive Layout** — ห้ามทำ Card-in-Card, ปล่อยเนื้อหาลงพื้นหลังตรง
- **ห้ามแสดงตัวเลขย่อย** — ห้ามพิมพ์เลขลำดับนำหน้าหัวข้อ แสดงแค่ชื่อ
- **ห้ามใส่ Header** — ระบบ `StandardHeader` จัดการให้อยู่แล้ว
- **โครงสร้าง Component**: `export default function ComponentName() { ... }`

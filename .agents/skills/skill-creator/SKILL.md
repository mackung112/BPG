# ตัวสร้างสกิล (Skill Creator)

ใช้สกิลนี้เมื่อต้องการช่วยเหลือผู้ใช้ในการสร้างหรืออัปเดตสกิลใหม่ในโปรเจกต์

## เมื่อใดควรใช้สกิลนี้
- เมื่อผู้ใช้ร้องขอให้สร้างสกิลใหม่ บูทสแตรปสกิล หรือติดตั้งคู่มือการทำงาน (Custom Skill) ในโปรเจกต์
- เมื่อมีการถามถึงวิธีการสร้างและลงทะเบียนสกิลตามโครงสร้างมาตรฐานของ Antigravity

## ขั้นตอนการสร้างสกิลใหม่

1. **รวบรวมข้อมูล**:
   - ถามผู้ใช้เกี่ยวกับชื่อสกิลที่ต้องการ (ควรเป็นภาษาอังกฤษ ตัวพิมพ์เล็ก และใช้เครื่องหมายยัติภังค์ `-` คั่น เช่น `code-reviewer`)
   - ถามคำอธิบายสั้นๆ ของสกิล (description) เพื่อระบุความสามารถและเงื่อนไขการใช้ของสกิล
   - ถามถึงข้อกำหนด คำสั่ง หรือแนวทางอื่นๆ ที่จำเป็นต้องระบุไว้ในคู่มือการใช้งานสกิล

2. **สร้างโฟลเดอร์ของสกิล**:
   - สร้างโฟลเดอร์ใหม่ภายใต้ไดเรกทอรี `.agents/skills/<skill-name>/`
   - หากมีการใช้สคริปต์ ตัวอย่าง หรือทรัพยากรเสริม ให้สร้างโฟลเดอร์ย่อย:
     - `scripts/` สำหรับสคริปต์การทำงาน
     - `examples/` สำหรับตัวอย่างการเรียกใช้งาน
     - `resources/` หรือ `references/` สำหรับเอกสารหรือเทมเพลต

3. **เขียนไฟล์ SKILL.md**:
   - สร้างไฟล์ `SKILL.md` ภายในโฟลเดอร์ที่สร้างขึ้น
   - เขียน YAML frontmatter ไว้ตอนเริ่มต้นไฟล์ในรูปแบบต่อไปนี้:
     ```yaml
     ---
     name: <skill-name>
     description: <คำอธิบายจุดประสงค์ของสกิลและเงื่อนไขการเรียกใช้>
     ---
     ```
   - เขียนเนื้อหาหลักของสกิล เช่น หัวข้อวัตถุประสงค์ เงื่อนไขการใช้งาน (When to use this skill) และคำแนะนำในการทำงาน (Instructions) เป็นภาษาไทยหรือภาษาอังกฤษตามที่ผู้ใช้ต้องการ

4. **ตรวจสอบการทำงาน**:
   - ตรวจสอบว่าโครงสร้างโฟลเดอร์และไฟล์ `SKILL.md` ถูกสร้างครบถ้วนถูกต้อง
   - ยืนยันกับผู้ใช้ว่าสกิลใหม่ถูกติดตั้งเรียบร้อยแล้ว และระบบจะตรวจหา (Auto-discover) สกิลนี้โดยอัตโนมัติในการรันครั้งถัดไป

# กฎการตอบกระชับ

1. **ทำแค่ที่สั่ง** — ไม่ต้องเพิ่มสิ่งที่ไม่ได้ขอ
2. **ห้ามอธิบายยืดยาว** — สรุปสั้นๆ ว่าทำอะไรไปแล้ว 1-2 บรรทัดพอ
3. **ไม่เข้าใจ ให้ถาม** — ถามก่อนทำเสมอถ้าข้อมูลไม่ชัดเจนหรือขาดหายไป
4. **ห้ามสรุปซ้ำ** — ไม่ต้องแสดงตารางสรุป ไม่ต้องเล่าซ้ำสิ่งที่เพิ่งทำ

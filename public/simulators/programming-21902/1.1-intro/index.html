💻 1.1.1 ภาษาคอมพิวเตอร์ คืออะไร?

สื่อกลาง

"สะพานเชื่อมความคิด"

ใช้สื่อสารระหว่างมนุษย์กับเครื่องจักร เพื่อถ่ายทอดลอจิกและแนวคิดให้คอมพิวเตอร์รับรู้และทำงานตามต้องการ

Human Idea ➔ Machine Action

ชุดคำสั่ง

"Instructions"

การเขียนโปรแกรมคือการเรียบเรียงชุดคำสั่งอย่างเป็นระบบ เพื่อแก้ปัญหาหรือสร้างสรรค์นวัตกรรมต่างๆ

print("Hello") ➔ Output

ตัวแปลภาษา

"Translator"

คอมพิวเตอร์เข้าใจแค่ 0 กับ 1 (Machine Code) จึงต้องมีตัวแปลภาษาเพื่อให้ทำงานได้จริง

Source Code ➔ 01010100
บุคคลสำคัญ (ฝั่งซอฟต์แวร์)

เอดา เลิฟเลซ (Ada Lovelace)

ได้รับการยกย่องว่าเป็น "โปรแกรมเมอร์คนแรกของโลก"

เธอเป็นผู้สร้างอัลกอริทึมแรกที่ออกแบบมาเพื่อให้เครื่อง Analytical Engine ประมวลผล เอดามองเห็นศักยภาพว่าเครื่องจักรไม่เพียงแต่ใช้คำนวณตัวเลขได้เท่านั้น แต่ยังสามารถจัดการสัญลักษณ์หรือแม้แต่สร้างเสียงดนตรีได้ หากมีชุดคำสั่งที่เหมาะสม

บุคคลสำคัญ (ฝั่งฮาร์ดแวร์)

ชาร์ลส แบบเบจ (Charles Babbage)

ได้รับการยกย่องว่าเป็น "บิดาแห่งคอมพิวเตอร์"

นักคณิตศาสตร์ชาวอังกฤษ ผู้ออกแบบและคิดค้นเครื่องคิดเลขเชิงกลที่เรียกว่า Difference Engine และต่อยอดไปสู่ Analytical Engine ซึ่งเป็นต้นแบบของสถาปัตยกรรมคอมพิวเตอร์ในปัจจุบัน (มีหน่วยประมวลผลและหน่วยความจำ)

1.1.3 องค์ประกอบของภาษาคอมพิวเตอร์

🔠

1. คำศัพท์ (Vocabulary)

คำสงวน (Keywords) ที่ภาษานั้นอนุญาตให้ใช้

if, for, while, print Keywords
📝

2. ไวยากรณ์ (Syntax)

กฎเกณฑ์การเรียงลำดับคำและการใส่เครื่องหมาย

print("Hello") Correct!
🧠

3. ความหมาย (Semantics)

ผลลัพธ์หรือพฤติกรรมที่เกิดขึ้นเมื่อรันโค้ด

3 + 5 ➔ 8

1.1.4 วิวัฒนาการของภาษาคอมพิวเตอร์

คลิกเลือกยุคเพื่อดูพัฒนาการของภาษา

ภาษาเครื่อง (Machine Language)

ประกอบด้วยเลข 0 และ 1 คอมพิวเตอร์เข้าใจได้ทันที แต่ยากต่อมนุษย์ ขึ้นอยู่กับฮาร์ดแวร์โดยตรง

ภาษาแอสเซมบลี (Assembly)

ใช้ตัวอักษรย่อ (Mnemonic) แทนเลข 0 และ 1 เช่น ADD, SUB ต้องใช้ Assembler แปลงกลับเป็นภาษาเครื่อง

ภาษาระดับสูง (High-Level)

ไวยากรณ์คล้ายภาษาอังกฤษและคณิตศาสตร์ (เช่น C, FORTRAN) ต้องใช้ Compiler หรือ Interpreter ในการแปล

ภาษาเชิงวัตถุ (Object-Oriented)

เน้นการแก้ปัญหาแบบวัตถุ (OOP) ช่วยลดความซ้ำซ้อนของโค้ด เช่น Java, C++, C#

ภาษาสมัยใหม่ (Modern & Natural)

เน้นความเรียบง่ายและคล่องตัวสูง มีไลบรารีรองรับ AI มหาศาล เช่น Python, JavaScript

1.1.5 ตัวแปลภาษา (Translator)

จำลองการทำงานของตัวดำเนินการ เพื่อดูว่าจะส่งผลลัพธ์ (ขั้นตอนการแปลโค้ด) อย่างไร

จำลอง Translator

# ตรวจสอบคำตอบผู้ใช้
if user_answer == correct_answer:
    print("True")
else:
    print("False")

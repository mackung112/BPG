# โฟลเดอร์สำหรับวางแผนการสอน (Lesson Plans)

โฟลเดอร์ `lesson-plans` นี้ถูกสร้างขึ้นมาเพื่อให้คุณครูใช้เป็นพื้นที่ส่วนตัวสำหรับ "ร่างโครงสร้างเนื้อหา" และ "วางแผนการสอน" ในแต่ละรายวิชาครับ (ไฟล์ในโฟลเดอร์นี้จะไม่มีผลต่อการทำงานของหน้าเว็บไซต์ เป็นเพียงกระดาษทดของคุณครูครับ)

## วิธีการใช้งาน
1. เมื่อคุณต้องการวางแผนวิชาใหม่ ให้สร้างไฟล์ `.md` (Markdown) ใหม่ขึ้นมาในโฟลเดอร์นี้ เช่น `math-m4.md`, `science-basic.md`
2. คุณสามารถคัดลอกโครงสร้างจากไฟล์ `template.md` ไปใช้เป็นแบบฟอร์มเริ่มต้นในการเขียนแผนได้เลยครับ
3. เมื่อวางแผนในไฟล์ `.md` จนตกผลึกแล้ว ค่อยนำเนื้อหา (ชื่อบทเรียน, ไฟล์ Simulator) ไปสร้างจริงในโฟลเดอร์ `public/simulators` และกรอกข้อมูลลงใน `src/data/curriculum.json` ในภายหลังครับ
